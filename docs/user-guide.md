## 5. Safety Guidelines

### 5.1 Critical Safety Warnings

#### ‚ö†Ô∏è Lethal Hazard Warning
**Pure caffeine is potentially lethal.** This application provides safety features, but user responsibility is critical.

#### Mandatory Safety Requirements
- **Precision Scale**: 0.001g accuracy minimum
- **Proper Equipment**: Lab-grade measuring tools
- **Safety Equipment**: Gloves, goggles, ventilation
- **Knowledge**: Understanding of chemical handling

### 5.2 EU Safety Limits

#### Caffeine Limits
- **Single Serving**: Maximum 80mg per serving
- **Daily Limit**: Maximum 400mg per day
- **Concentration**: Maximum 150mg per 100ml

#### Ingredient Restrictions
- **Titanium Dioxide (E171)**: Banned in EU since 2022
- **Preservatives**: Limited concentrations
- **Sweeteners**: Approved types only

### 5.3 Safe Handling Procedures

#### Measurement Safety
1. **Always use precision scale** (0.001g accuracy)
2. **Double-check measurements** before mixing
3. **Never eyeball** caffeine or concentrates
4. **Label all containers** clearly

#### Storage Requirements
- **Caffeine**: Lock away from children/pets
- **Chemical concentrates**: Proper ventilation
- **Finished products**: Refrigeration for freshness
- **Documentation**: Keep ingredient safety data sheets

## 7. Troubleshooting

### 7.1 Common Issues

#### Application Won't Load
**Symptoms**: Blank screen, infinite loading
**Solutions**:
1. Check internet connection
2. Clear browser cache
3. Disable browser extensions
4. Try incognito/private mode
5. Update browser to latest version

#### Calculator Not Working
**Symptoms**: Error messages, incorrect calculations
**Solutions**:
1. Refresh the page
2. Clear application data
3. Check input values are valid
4. Disable ad blockers
5. Try different browser

#### Offline Mode Issues
**Symptoms**: Can't access offline content
**Solutions**:
1. Ensure app was used online first
2. Check if Service Worker is registered
3. Clear browser data and re-sync
4. Update to latest version

### 7.2 Performance Issues

#### Slow Loading
**Possible Causes**:
- Slow internet connection
- Too many browser tabs open
- Outdated browser
- Browser extensions interfering

**Solutions**:
1. Close unnecessary browser tabs
2. Update browser
3. Disable unnecessary extensions
4. Use faster internet connection

#### High Memory Usage
**Solutions**:
1. Close other applications
2. Restart browser
3. Clear browser cache
4. Update to latest browser version

### 7.3 Error Messages

#### "Age Verification Required"
- **Cause**: First-time visit or cleared data
- **Solution**: Complete age verification process

#### "Calculation Error"
- **Cause**: Invalid input values
- **Solution**: Check input ranges and formats

#### "Safety Limit Exceeded"
- **Cause**: Calculator detected unsafe levels
- **Solution**: Reduce caffeine concentration or volume

#### "Offline Mode Active"
- **Cause**: No internet connection
- **Solution**: Connect to internet for full functionality

## 9. Frequently Asked Questions

### 9.1 General Questions

**Q: Is this application free to use?**
A: Yes, the application is completely free. We may include affiliate links to suppliers, but there's no cost to use the app.

**Q: Do I need to create an account?**
A: No account required. All data is stored locally in your browser.

**Q: Is my data safe?**
A: Yes. We follow GDPR guidelines and don't store personal data. All calculations are done locally.

**Q: Can I use this offline?**
A: Yes, with limited functionality. Core features work offline after initial load.

### 9.2 Safety Questions

**Q: Is it safe to make energy drinks at home?**
A: When following our safety guidelines and EU regulations, home energy drink creation can be safe. However, you must use proper equipment and follow safety procedures.

**Q: What if I exceed the safety limits?**
A: The calculator will warn you if you exceed EU safety limits. Never ignore these warnings.

**Q: Can I give energy drinks to minors?**
A: No. Our application requires 18+ age verification and we strongly advise against giving energy drinks to minors.

### 9.3 Technical Questions

**Q: Why do I need a 0.001g scale?**
A: Caffeine is lethal in small amounts. Kitchen scales aren't precise enough for safe measurements.

**Q: Can I use different suppliers?**
A: Yes, but verify ingredient quality and purity. We recommend EU-approved suppliers.

**Q: How accurate are the calculations?**
A: Calculations are precise when proper equipment is used. Always double-check measurements.

### 9.4 Troubleshooting Questions

**Q: The calculator shows an error. What should I do?**
A: Check that your inputs are valid numbers within reasonable ranges. Try refreshing the page.

**Q: Can I save my calculations?**
A: Currently, calculations aren't saved automatically. You can bookmark results or use the recipe feature.

**Q: How do I update the application?**
A: Updates are automatic. Look for "Update Available" notifications.
