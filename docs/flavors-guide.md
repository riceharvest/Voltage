## üé≠ Creating Custom Flavors

### Base Flavor Selection

#### Starting Points
1. **Choose a Base**: Classic, Zero, or Plain
2. **Select Primary Flavor**: Choose main taste profile
3. **Add Secondary Flavors**: Enhance with complementary tastes
4. **Adjust Ratios**: Fine-tune flavor intensity

#### Compatibility Guidelines
- **Citrus + Berry**: Bright and sweet combinations
- **Tropical + Vanilla**: Smooth and creamy tropical
- **Classic + Citrus**: Familiar taste with twist
- **Unique + Sweet**: Balance bold flavors

### Ratio Guidelines

#### Primary Flavor (70-80%)
- **Concentrate**: 0.1-0.5ml per 100ml base
- **Extract**: 1-5ml per 100ml base
- **Oil**: 0.5-2ml per 100ml base
- **Natural**: 5-15ml per 100ml base

#### Secondary Flavor (20-30%)
- **Concentrate**: 0.05-0.25ml per 100ml base
- **Extract**: 0.5-2.5ml per 100ml base
- **Oil**: 0.25-1ml per 100ml base
- **Natural**: 2.5-7.5ml per 100ml base

### Testing and Refinement

#### Small Batch Testing
1. **Start Small**: Test with 50-100ml batches
2. **Document Changes**: Keep detailed notes
3. **Taste at Intervals**: Check flavor development over time
4. **Adjust Gradually**: Make small changes between tests

#### Batch Scaling
- **Proportional Scaling**: Maintain ingredient ratios when scaling
- **Rounding Considerations**: Practical measurement adjustments
- **Cost Analysis**: Calculate per-serving costs at different scales
- **Quality Assurance**: Ensure consistency across batch sizes

## üîß Troubleshooting Flavor Issues

### Common Problems

#### Flavor Too Weak
**Symptoms**: Barely detectable flavor
**Causes**: 
- Insufficient concentrate quantity
- Poor quality ingredients
- Wrong ratio calculations
- Inadequate mixing time

**Solutions**:
- Increase concentrate by 25%
- Check ingredient expiration dates
- Verify measurements with precision scale
- Allow longer mixing/aging time

#### Flavor Too Strong
**Symptoms**: Overpowering, unpleasant taste
**Causes**:
- Excessive concentrate quantities
- Low-quality concentrates
- Wrong ingredient ratios
- Poor mixing technique

**Solutions**:
- Reduce concentrate by 50%
- Dilute with additional base
- Start over with correct ratios
- Mix more thoroughly

#### Off-Taste or Chemical
**Symptoms**: Unpleasant chemical aftertaste
**Causes**:
- Expired ingredients
- Poor quality sources
- Wrong storage conditions
- Contamination

**Solutions**:
- Replace all ingredients
- Source from verified suppliers
- Check storage requirements
- Ensure clean equipment

#### Inconsistent Results
**Symptoms**: Different taste between batches
**Causes**:
- Inconsistent measurements
- Ingredient quality variation
- Temperature differences
- Poor mixing technique

**Solutions**:
- Use precision equipment consistently
- Source from same suppliers
- Control mixing temperature
- Standardize mixing procedure
